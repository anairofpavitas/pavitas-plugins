# Connectors

## Connected tools

This plugin has been customized for Pavitas Productions with the following tool mapping:

| Category | Tool(s) | Notes |
|----------|---------|-------|
| Chat | Slack, Discord, iMessage, Telegram | Multiple chat tools — Slack and Discord are primary |
| Email | Gmail | Via Google Workspace (pavi@paviproczko.com) |
| Cloud storage | Google Drive, Box | Google Drive for docs, Box for file storage |
| Knowledge base | Notion | Central hub — AI Biz OS |
| Project tracker | Notion | Notion databases for project/task tracking |
| CRM | Notion | Lightweight client tracking within Notion |
| Office suite | Google Workspace | Docs, Sheets, Slides |

## Adding more sources

To add a new source, update the `.mcp.json` file with the MCP server configuration. The search and digest commands will automatically detect and include new sources.
