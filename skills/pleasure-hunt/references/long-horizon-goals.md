# Long-horizon goals

A living ledger of standing sexual goals. **Not a checklist.** None of these have a finish line — they are practices to deepen, revisit, and specify further over time.

This file does two jobs:

1. **Catalog** — the menu of goals a hunt can be built around, or fold a thread of.
2. **Ledger** — a running, dated log under each goal of what's been done and where, so future hunts can nudge toward neglected goals and deepen ones already in rotation.

---

## How to use the ledger

**At intake.** Scan the goals. If one fits the hunt's who / flavor / boundaries and its log shows no recent attention, offer it as a thread — one line, never forced. If a goal's log shows recent activity, consider *deepening* instead of repeating.

**Deepening, not completing.** When a goal has entries, the next hunt's version should advance — a longer hold, a bolder location, a higher tier, less assistance, a new variation. Ask "what would make this more than last time?" — never "is this done?"

**After a hunt.** If the hunt touched a goal, offer to add a dated line to that goal's log. Keep it brief and factual. The user can decline. The log is committed to the repo; it is what lets the next hunt build on the last.

**Log entry format:**

```
- YYYY-MM-DD — what happened, where. Optional note for next time.
```

**Alone vs. together.** Most goals split into an *alone* variant (the user solo, or out without Colin) and a *together* variant (with Colin). They are different practices — log and deepen them separately.
