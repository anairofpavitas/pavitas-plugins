# End of Day Review

Wrap up the day and set up tomorrow.

## Execution Steps

1. **Check today's briefing** — pull the Littlebird Log entry from this morning
2. **Review the 3-3-3** against what actually happened:
   - Which deep focus items were completed?
   - Which quick wins got done?
   - Which habits/joy items happened?
3. **Check Notion Tasks** — any items completed today? Any new items added?
4. **Check email** — any loose threads from today that need tomorrow follow-up?
5. **Check calendar** — what's on tomorrow?

## Present Review

```
🌙 END OF DAY — [Date]

✅ COMPLETED
- [Items that got done]

➡️ CARRIES FORWARD
- [Items that didn't get done — auto-add to tomorrow's briefing]

📝 NOTES
- [Anything worth recording for future reference]

🔮 TOMORROW PREVIEW
- [Key items for tomorrow]
```

## After Review

- Update today's Littlebird Log entry with completion status
- Carry forward incomplete items (they'll appear in tomorrow's briefing carryforward section)
- If Pavi has been working 45+ minutes continuously, remind him to take a break before wrapping up

## Decision Framework Integration

If patterns emerge (e.g., deep focus items consistently not getting done, same task carrying forward 3+ days), flag it:
- "This is the third day [task] has carried forward. Worth either doing it, delegating it, or dropping it?"
