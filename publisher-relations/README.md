# Publisher Relations Plugin

Manage all publisher communications, invoicing, and project tracking for Pavitas Productions LLC.

## Commands

| Command | What it does |
|---------|-------------|
| `/publisher-relations:intake` | Process a publisher email — extract actions, deadlines, match to Notion project |
| `/publisher-relations:invoice` | Generate branded PDF invoice with stored publisher billing info |
| `/publisher-relations:status` | Pull all active projects for a publisher with deadlines and completion % |
| `/publisher-relations:reply` | Draft professional reply in Biz Mode (full review before sending) |

## Publisher Profiles

Stores billing info, contacts, payment terms, and delivery preferences for:
- Aethon (Julie Holloway)
- Podium
- Recorded Books
- Blackstone
- Portal Books / VoiceWorks Audio (Daniel Baker)
- Audible direct

## Connectors

- **Gmail** — Publisher email communications
- **Notion** — Audiobook Projects database for project context
- **Box** — File delivery

## References

- `business-documentation` skill — logo, branding, invoice formatting
- `design-elevation` skill — professional document output
