# Scene Analysis

Evaluate a scene using Story Grid methodology.

## Input
- Scene text (pasted, uploaded, or referenced by chapter)
- Genre context (if not obvious from the text)

## Analysis Framework: 5 Commandments

For each scene, evaluate:

### 1. Inciting Incident
- Is there a clear event that upsets the balance?
- Causal or coincidental?
- Does it create a question the scene must answer?

### 2. Turning Point Progressive Complication
- Does the complication force a dilemma?
- Is it the most extreme complication in the scene (not just any complication)?
- Action or revelation?

### 3. Crisis
- Is there a clear best bad choice or irreconcilable goods dilemma?
- Is the character forced to make an active decision?
- Are both options genuinely difficult?

### 4. Climax
- Does the character take action based on their crisis decision?
- Is it an irreversible action?
- Does it reveal character (what they do under pressure)?

### 5. Resolution
- What's the new normal after the climax?
- Has the value charge shifted? (positive → negative, or vice versa)
- Does it set up the next scene's inciting incident?

## Value Shift
- Identify the core value at stake (life/death, freedom/oppression, love/hate, etc.)
- Track the shift from beginning to end of scene
- Note the polarity: +, -, ++, --, +/- (ironic)

## Genre-Specific Checks

### LitRPG / Progression Fantasy (Jim & Percy)
- Does the scene advance the protagonist's progression?
- Are system/game mechanics integrated into the narrative (not just info-dumped)?
- Does the power gain come with meaningful cost or consequence?
- For combat scenes: does the outcome depend on the character's choices, not just stats?

### Obligatory Scene Tracker
If analyzing a full story/act, check for missing obligatory scenes:
- Hero at the mercy of the villain
- All is lost moment
- Core event of the global genre
- Point of no return

## Output

```
📖 SCENE ANALYSIS

Value: [Core value] — [Starting charge] → [Ending charge]
Shift: [+/-/++/--/ironic]

1. INCITING INCIDENT: [Present/Weak/Missing]
   [Brief description]

2. TURNING POINT: [Present/Weak/Missing]
   Type: [Action/Revelation]
   [Brief description]

3. CRISIS: [Present/Weak/Missing]
   Type: [Best bad choice / Irreconcilable goods]
   [Brief description]

4. CLIMAX: [Present/Weak/Missing]
   [Brief description]

5. RESOLUTION: [Present/Weak/Missing]
   [Brief description]

💡 VERDICT: [Working / Needs revision / Not yet a scene]
[One specific, actionable suggestion]
```

## Rules
- Be honest. If a scene isn't working, say so — but constructively.
- One actionable suggestion, not a rewrite plan.
- Use Story Grid terminology naturally.
- Reference the 20 Core Scenes checklist when evaluating broader structure.
