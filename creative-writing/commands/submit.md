# Submission Prep

Prepare a scene or piece for Joanne Mitchell's Story Grid mentorship review.

## Execution

1. **Check last submission date**
   - Search Notion Littlebird Log for "Joanne" or "submission" entries
   - Calculate days since last submission
   - Weekly cadence is the goal — flag if overdue

2. **Format the submission**
   - Clean prose through the humanize-prose skill (anti-AI-isms)
   - Standard manuscript format unless Joanne prefers otherwise
   - Include scene context: where it falls in the narrative, what precedes it
   - Note which of the 5 Commandments Pavi is most uncertain about

3. **Self-assessment header**
   - What Pavi thinks is working
   - What Pavi thinks needs help
   - Specific questions for Joanne

4. **Track in Notion**
   - Log submission date in Littlebird Log
   - Note what was submitted
   - Set reminder to check for Joanne's response in 3-5 days

## Output

Present the formatted submission for Pavi's review before sending.

## Rules
- Never send directly — always present for review
- If it's been 10+ days since last submission, include a gentle nudge but no guilt
- If Pavi is stuck and has nothing to submit, suggest the `/creative-writing:stuck` command instead
