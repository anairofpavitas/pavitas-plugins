# Fiber Arts & Content Plugin

Crochet project documentation and Instagram content creation for @pavi.spins.yarns.

## Commands

| Command | What it does |
|---------|-------------|
| `/fiber-arts-content:pattern` | Transcribe pattern from YouTube/web into standard crochet format (PDF) |
| `/fiber-arts-content:log` | Save pattern or project update to Notion + crochet/ folder |
| `/fiber-arts-content:post` | Draft Instagram content with Spinning Yarns brand voice |
| `/fiber-arts-content:project` | Track active crochet project status and research |

## Connectors

- **Notion** — Littlebird Log for pattern storage and project tracking
