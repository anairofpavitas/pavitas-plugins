# Fiber Arts Log

Quick save for pattern documents and project progress updates.

## Usage

### Pattern Storage
"Log this pattern" → Save formatted pattern to:
1. Notion Littlebird Log (with appropriate tags)
2. crochet/ folder organized by article type

### Project Update
"Log crochet update" → Create Littlebird Log entry with:
- Which project (waistband, offset cardigan, landscape blanket, etc.)
- What was accomplished
- Any decisions made (stitch choices, color changes, modifications)
- What's next
